# JuegoPreguntasYRespuestas

Proyecto desarrollado en C# con Windows Forms (.NET) y MySQL como base de datos.

Descripción:
Aplicación de escritorio que implementa un juego de trivia con interfaz gráfica personalizada utilizando un lienzo (sin controles tradicionales como botones).
El jugador selecciona una categoría y responde 10 preguntas en orden aleatorio. Al finalizar, se muestra la puntuación y se guarda la partida en la base de datos.

Características principales:
- Interfaz gráfica dibujada con Graphics
- Mínimo 5 categorías
- Mínimo 10 preguntas por categoría (50 en total)
- 50% preguntas de opción múltiple (texto)
- 50% preguntas con selección de respuesta con imagen
- Guardado completo en base de datos
- Historial de partidas

Tecnologías utilizadas:
- C#
- Windows Forms (.NET)
- MySQL
- MySQL Workbench
- Git & GitHub

Configuración inicial:
1. Crear la base de datos en MySQL.
2. Configurar la cadena de conexión en el archivo ConexionBD.cs.
3. Ejecutar el proyecto en Visual Studio.
